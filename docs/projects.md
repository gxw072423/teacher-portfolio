# 项目作品

## 毕业设计：校园二手交易平台

<div class="proj" markdown>

<div class="stack">Figma / HTML / CSS / JavaScript</div>

这是我花心思最多的项目。为大学校园做了一款二手交易平台的 UI 设计和前端实现。

从用户调研开始，画信息架构、做线框图，再到视觉设计和高保真原型——在 Figma 里反复打磨。设计完成后，用原生 HTML/CSS/JS 实现了商品列表、详情页、发布页、聊天界面等核心页面，也做了响应式适配。

具体产出：40+ 页 Figma 高保真设计稿、组件库和设计规范、核心页面代码、手机和桌面端适配。

**学到的**：设计不是画图，是解决问题。好的 UI 让用户不用思考就知道下一步该做什么。
